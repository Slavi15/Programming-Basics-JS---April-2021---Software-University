# Even or Odd
---

## Description
Write a function which gets 1 number as an argument and prints "even" **or** "odd"

## Input
1. Number - whole num [0 - 10000]

## Output
1. even **or** odd

## Input & Output examples

|  Input  |  Output  |
| ------- | :------: |
|  [ 2 ]  |   even   |
|  [ 25 ] |   odd    |
