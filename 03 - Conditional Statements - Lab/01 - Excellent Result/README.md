# Excellent Result
---

## Description
Write a function gets a number as an argument and prints **"Excellent!"** if the number is 5.5 or greater

## Input
1. Number - real num [2 - 6]

## Output
1. **"Excellent!"** or **" "**

## Input & Output examples

|  Input  |   Output   |
| ------- | :--------: |
|  [ 6 ]  | Excellent! |
|  [ 5 ]  |            |
