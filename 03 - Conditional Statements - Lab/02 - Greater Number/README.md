# Greater Number
---

## Description
Write a function which gets 2 numbers as arguments and prints out the bigger one

## Input
1. Number - real num [0 - 10000]
2. Number - real num [0 - 10000]

## Output
1. Number - real num [0 - 10000]

## Input & Output examples

|    Input     |  Output |
| ------------ | :-----: |
|   [ 5, 3 ]   |    5    |
|  [ 10, 10 ]  |    10   |
