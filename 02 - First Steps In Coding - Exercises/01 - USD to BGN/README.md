# USD to BGN
---

## Description
Write a function which converts USD to BGN. Use fixed currency course between USD and BGN. (1 USD = 1.79549 BGN)

## Input
1. Number

## Output
1. Number in BGN currency

## Input & output examples

|  Input   |   Output   |
| -------- | :--------: |
|  [ 22 ]  |  39.50078  |
