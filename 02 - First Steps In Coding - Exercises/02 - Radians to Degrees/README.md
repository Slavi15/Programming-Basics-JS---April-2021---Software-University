# Radians to Degrees
---

## Description
#### Write a function which gets angle in radians and converts it to degrees. Use the formula.
#### Use this formula:    degree = (radian * 180) / PI

## Input
#### 1. Angle in radians [ Number ]

## Output
#### 1. Angle in degrees [ Number ]

## Input & Output examples

|  Input   |  Output  |
| -------- | :------: |
|  3.1416  |   180    |
